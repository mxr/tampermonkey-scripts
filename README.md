# tampermonkey-scripts

Personal Tampermonkey scripts for daily browser automation.

## Scripts

- `scripts/etrade-login-security-code.user.js`: Automatically enables "Use security code" on the E*TRADE login experience.

## Usage

1. Install [Tampermonkey](https://www.tampermonkey.net/).
2. Open the script file on GitHub.
3. Click `Raw` to open the Tampermonkey install page.
4. Click `Install`.

## Install In Firefox

1. In Firefox, install the Tampermonkey add-on: <https://addons.mozilla.org/firefox/addon/tampermonkey/>.
2. Open this script page on GitHub: <https://github.com/mxr/tampermonkey-scripts/blob/main/scripts/etrade-login-security-code.user.js>.
3. Click `Raw`.
4. Tampermonkey opens an install tab for the script. Click `Install`.
5. Visit `https://us.etrade.com/home/welcome-back` or `https://us.etrade.com/etx/pxy/login`.
